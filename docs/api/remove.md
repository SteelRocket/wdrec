# Path
http://127.0.0.1:{port}/remove

# Usage

Used to remove a replace word from wdrec

# Parameters

| parameter |    type    |         description          | incompatible |
|:---------:|:----------:|:----------------------------:|:------------:|
|   word    | tag string | The source word to remove by |      id      |
|    id     |    int     |     The id to remove by      |     word     |
